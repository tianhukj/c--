### 💻 如何打开Windows管理员终端

> **Windows系统打开管理员终端的方法：**
> 1. 使用快捷键 `Win + X`
> 2. 在弹出的菜单中选择:
>    - "Windows PowerShell (管理员)" 
>    - "Windows Terminal (管理员)"
>    - "终端(管理员)"
>    (根据系统版本可能显示不同选项)

#### Windows 安装特性:
- 🔍 自动检测并使用 PowerShell 7（如果可用）
- 🛡️ 通过 UAC 提示请求管理员权限
- 📝 如果没有 PS7 则使用 Windows PowerShell
- 💡 如果提权失败会提供手动说明
